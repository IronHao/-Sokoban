## 简单介绍
    一个简单的推箱子游戏，通过WASD操作人物，将箱子全部推至指定位置后完成游戏。有以下几个功能:
- 有一个主操作界面
- 自带三档难度的地图
- 可自定义地图，并附带简单的错误检测（如不能同时拥有两个人物，箱子数量要与目标数量一致等）
- 可以随时保存/读入地图，目前最多为3份
- 可以记录步数
- 拥有撤回操作
- 有一定的异常报错处理（不完善，有BUG, 咕了）

## 使用方法
